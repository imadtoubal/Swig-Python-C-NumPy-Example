# SWIG
This is an example of using C++ to write cutsom functions to extend numpy through
the SWIG interface. The example in this repo calculates the root mean square
(RMS) of a sequence of numbers.
## Getting started

Build
```bash
make
```

Test
```bash
make test
``` 

Clean
```bash
make clean
```
